# 手相鑑定LP かんたん公開手順

さくらやVercelが難しい方向け。以下の2つから選んでください。

---

## 方法A: Netlify Drop（いちばん簡単・ドラッグ＆ドロップ）

**アカウント不要**で、フォルダをドロップするだけで公開できます。

### 手順

1. **このページを開く**  
   [https://app.netlify.com/drop](https://app.netlify.com/drop)

2. **フォルダをドラッグ＆ドロップ**
   - Finder で `/Users/yui/Documents/cursor/tesou-lp` を開く
   - **index.html と images フォルダが入ったフォルダ**を、Netlify の画面にドラッグ＆ドロップ  
   （`tesou-lp` フォルダごとドロップでOK）

3. **完了**
   - 数秒で「Site is live」と表示され、URL（例: `https://xxxx.netlify.app`）が発行されます
   - そのURLをクリックして表示を確認

4. **（任意）アカウント作成でURLを保持**
   - アカウントなしだとURLは一時的になります
   - 「Sign up」で無料アカウントを作ると、同じURLを継続して使えます

---

## 方法B: GitHub Pages（すでにGitHubにあります）

コードはすでに GitHub にあるので、**設定を1つ変えるだけ**で公開できます。

### 手順

#### ステップ1: リポジトリを開く（404になる場合の対処）

1. **[github.com](https://github.com)** にアクセス
2. **ログイン**していることを確認（未ログインの場合、右上「Sign in」からログイン）
3. リポジトリを開く  
   次のどちらかでアクセス：
   - **方法①**: 左上の GitHub ロゴをクリック → 「Your repositories」→ **tesou-lp** をクリック
   - **方法②**: アドレス欄に `https://github.com/Yp465-ck/tesou-lp` を入力して直接開く（ログイン済みであること）

※ **404 が出る場合**: ログインしていない、または別アカウントでログインしている可能性があります。正しいアカウントでログインし、「Your repositories」から tesou-lp を探してください。

※ **リポジトリが「Private」の場合**: 無料の GitHub Pages を使うには、**Settings** → **General** → 一番下の「Danger Zone」→ **Change repository visibility** → **Make public** で公開に変更してください。

---

#### ステップ2: GitHub Pages を有効にする

1. リポジトリの画面で、画面上方の **「Settings」** タブをクリック
2. 左側のメニューで **「Pages」** をクリック  
   （「Code and automation」の下にある場合があります）
3. **「Build and deployment」** の欄で：
   - **Source**: 「Deploy from a branch」を選択
   - **Branch**: 「main」を選び、**Folder** で 「/ (root)」を選択
   - **Save** をクリック

---

#### ステップ3: 公開を待つ

- 1〜2分ほど待つと、「Your site is live at」の下に公開URLが表示されます
- 初回は数分かかることがあります

### 公開URL

**https://yp465-ck.github.io/tesou-lp/**

（アカウント名が `Yp465-ck` の場合。別のアカウントの場合は `Yp465-ck` の部分があなたのユーザー名になります）
