# 手相鑑定LP デプロイ手順

静的HTMLのランディングページです。Vercel または GitHub Pages で公開できます。

---

## 方法1: Vercel でデプロイ（推奨）

### 1. GitHub にプッシュ

```bash
cd /Users/yui/Documents/cursor/tesou-lp
git init
git add .
git commit -m "手相鑑定LP 初回コミット"
git branch -M main
```

GitHub で新しいリポジトリ（例: `tesou-lp`）を作成したあと：

```bash
git remote add origin https://github.com/あなたのユーザー名/tesou-lp.git
git push -u origin main
```

### 2. Vercel でインポート

1. **[vercel.com](https://vercel.com)** にログイン（GitHub アカウントで可）
2. **Add New** → **Project** をクリック
3. **Import Git Repository** で `tesou-lp` を選択
4. **Deploy** をクリック
5. 1〜2分待つと、公開URL（例: `https://tesou-lp-xxx.vercel.app`）が発行されます

※ 環境変数は不要です。そのままデプロイ可能です。

---

## 方法2: Vercel CLI でデプロイ（Node.js がある場合）

```bash
cd /Users/yui/Documents/cursor/tesou-lp
npx vercel
```

初回はログインやプロジェクト設定の案内に従います。その後、表示されるURLが公開先です。

---

## 方法3: GitHub Pages

1. 上記の通り GitHub にプッシュ
2. リポジトリの **Settings** → **Pages**
3. **Source** で `main` ブランチを選択
4. **Save** 後、数分で `https://あなたのユーザー名.github.io/tesou-lp/` で公開されます

※ GitHub Pages の場合、リポジトリ名がURLに含まれます。
