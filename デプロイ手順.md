## あと2分：Vercel で公開するだけ

### 1. このリンクを開く

**[→ Vercel でインポートする（クリック）](https://vercel.com/new/clone?repository-url=https://github.com/Yp465-ck/tesou-lp)**

### 2. GitHub でログイン

「Continue with GitHub」をクリックし、必要なら許可します。

### 3. Deploy をクリック

環境変数は不要です。そのまま **Deploy** をクリックしてください。  
1〜2分待つと、**Visit** ボタンから公開URLが開きます。

## 方法3: GitHub Pages

1. 上記の通り GitHub にプッシュ
2. リポジトリの **Settings** → **Pages**
3. **Source** で `main` ブランチを選択
4. **Save** 後、数分で `https://あなたのユーザー名.github.io/tesou-lp/` で公開されます

※ GitHub Pages の場合、リポジトリ名がURLに含まれます。
