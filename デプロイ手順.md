## 方法4: さくらサーバー（yui-uranai.sakura.ne.jp）へアップロード

独自ドメインで公開する場合の手順です。**kantei-menu** サブフォルダに配置します。

**公開URL: https://yui-uranai.sakura.ne.jp/kantei-menu/**

### アップロードするファイル

| ローカル | サーバー（kantei-menu フォルダ内） |
|----------|-----------------------------------|
| `index.html` | `index.html` |
| `images/profile.png` | `images/profile.png` |
| `images/hero-mystic.png` | `images/hero-mystic.png` |

`images` フォルダごとアップロードし、`kantei-menu` 直下に `images` として配置してください。

### FTPソフト（FFFTP / FileZilla など）での手順

1. **FTPでさくらに接続**
   - ホスト: `yui-uranai.sakura.ne.jp`（またはコントロールパネルに記載のFTPサーバー名）
   - ユーザー名・パスワード: さくらコントロールパネルで確認

2. **kantei-menu フォルダに移動**  
   - Web公開フォルダ（例: `www`）を開く  
   - `kantei-menu` フォルダを作成（初回のみ）し、その中に入る  
   ※ さくらコントロールパネル「ドメイン/SSL」→「基本設定」で公開フォルダを確認

3. **ファイルをアップロード**
   - `index.html` → `kantei-menu` フォルダの直下
   - `images` フォルダ → `kantei-menu` フォルダの直下に `images` としてそのまま配置

4. **完了**  
   `https://yui-uranai.sakura.ne.jp/kantei-menu/` でアクセス可能になります。

### コマンドで一括アップロード（lftp 使用）

`lftp` が入っている場合、`deploy-sakura.sh` で一括アップロードできます。デフォルトで `www/kantei-menu` へアップロードします。

```bash
cd /Users/yui/Documents/cursor/tesou-lp
export SAKURA_USER=あなたのFTPユーザー名
export SAKURA_PASS=パスワード  # 省略すると実行時に入力
# SAKURA_REMOTE_DIR はデフォルトで www/kantei-menu
chmod +x deploy-sakura.sh
./deploy-sakura.sh
```

※ 初回は `www` 内に `kantei-menu` フォルダがない場合、FTPソフトで事前に作成してください。  
※ `lftp` は `brew install lftp` でインストール可能です。
